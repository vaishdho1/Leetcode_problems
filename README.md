# Leetcode_problems
## Solutions to some of the leetcode problems solved


#### 1) Path_sum_II 
##### Given the root of a binary tree and an integer targetSum, 
  return all root-to-leaf paths where the sum of the node values in the path equals targetSum.
###### One approach using DFS ~ O(m+n)

#### 2) Longest_Increasing_Subsequence
##### Given an integer array nums, return the length of the longest strictly increasing subsequence.
###### Approach1: Using dynamic programming O(n*n)
###### Approach 2: Using binary search O(nlogn)

#### 3) Edit_distance
##### Given two strings word1 and word2, return the minimum number of operations required to convert word1 to word2.
###### Approach 1: Dynamic programming using space - O(m*n)
###### Approach 2: Dynamic programming using space - O(m), m>=n
